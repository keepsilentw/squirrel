#git config

* git status --> git st
    - `git config --global alias.st status`

* git checkout --> git co
    - `git config --global alias.co checkout`

* git commit --> git cm
    - `git config --global alias.cm commit`

* git branch --> git br
    - `git config --global alias.br branch`

* git merge --> git mg
    - `git config --global alias.mg merge`

* git stash --> git ss
    - `git config --global alias.ss stash`
