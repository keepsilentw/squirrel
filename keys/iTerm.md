#iTerm-keys

```
⌘: command
⌥: option/alt
⇧: shift
⌃: control
```

	1.⌘ + Click     可以打开文件，文件夹和链接
	
	2.⌘ + n         新建窗口
	
	3.⌘ + t         新建标签页
	
	4.⌘ + w         关闭当前页
	
	5.⌘ + 数字 & ⌘ + 方向键：切换标签页
	
	6.⌥⌘ + 数字     切换窗口
	
	7.⌘ + enter     切换全屏
	
	8.⌘ + d         左右分屏
	
	9.⇧⌘ + d        上下分屏
	
	10.⌘ + ;        自动补全历史记录
	
	11.⇧⌘ + h       自动补全剪贴板历史
	
	12.⌥⌘ + e       查找所有来定位某个标签页
	
	13.⌘ + r & ⌃ + l  清屏
	
	14.⌘ + /        显示光标位置

	15.⌘ + f        查找，然后用 tab 和 ⇧ + tab 可以向右和向左补全，补全之后的内容会被自动复制， 还可以用 ⌥ + enter 将查找结果输入终端
