#nginx-keys

* nginx                     启动
* nginx -c                  指定配置文件
* nginx -s quit             退出
* nginx -s stop             停止
* nginx -s reload           重载
* nginx -t                  检查配置文件
* nginx -v                  查看版本
* nginx -V                  查看版本详细
* ps -ef | grep nginx       查询nginx主进程号
* kill -TERM 进程号          快速停止
* pkill -9 nginx            强制停止nginx
* lsof -i:80				查看占用80端口的进程
